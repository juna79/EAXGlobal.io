# EAX Global

Environmental Asset Exchange (EAX)

A premium landing page for EAX Global — a platform focused on transparent infrastructure for environmental asset markets including carbon credits, biodiversity assets and verified environmental impact.

## 🌍 Vision

EAX Global is building market infrastructure for:

- Carbon markets
- Biodiversity assets
- Environmental finance
- Verified impact
- Transparent market access

The platform is designed to connect credible environmental projects with global buyers, investors and institutions.

# 🛠 Deployment

## 1. Push to GitHub

Upload:

- index.html
- README.md

to your repository.

---

## 2. Deploy with Netlify

Connect the GitHub repository to Netlify.

Netlify will automatically deploy the website.

---

## 3. Connect Domain

Add custom domain:

eaxglobal.io

inside Netlify Domain Settings.
